# DSA_Project

**Requirements**

Before running the project, ensure the following software and libraries are installed on your system:

    C Compiler: GCC is recommended.
    OpenSSL Library: Required for SHA256 hashing.

<b>Installation<b>

    cd DSA_Project
    ./run.sh
